\# BRAVO - Sign Language + Speech Recognition App



\## Features

\- Real-time hand sign recognition (OpenCV + Teachable Machine)

\- Speech-to-text using Faster-Whisper

\- CustomTkinter GUI



\## Setup



1\. Clone repo:

git clone https://github.com/yourname/project.git

cd project



2\. Install dependencies:

pip install -r requirements.txt



3\. Install FFmpeg:

https://ffmpeg.org/download.html



5\. You can run this project without a virtual environment, but using one is recommended to avoid dependency issues.

&#x20;## Setup (Recommended)



\- Create a virtual environment:

python -m venv venv



\- Activate it:



Windows:

venv\\Scripts\\activate



Mac/Linux:

source venv/bin/activate





4\. Run:

python main.py
